# SpeedSelfie

## Initialisation
```sh
$ php bin/console d:d:c
$ php bin/console d:s:c
$ php bin/console import:users equipe.csv
$ php bin/console import:questions questions.csv
```

## Formats

### Users
```
Firstname;Lastname;Username;Password
```
### Questions
```
Type;Question
```
Type allow *solo* or *multi* .
